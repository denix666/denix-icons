denix-icons
===========

denix-icons - is a set of icons for DeniX-OS.


Installation
============

**Howto build RPM from source:**

Clone my git repository and run the build script:

```vim
$mkdir git-repos
$cd git-repos
$git clone https://github.com/denix666/denix-icons.git
$cd denix-icons
$./build_denix-icons.sh
```
